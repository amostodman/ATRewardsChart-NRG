Energi (NRG) Rewards Charts

How to use:
  1. Select your reward data file (normally reward_data.csv)
    (An example reward data file: reward_data_example.csv)
  2. Click create charts
    Tip: Highlight part of a chart to zoom


ATRewardsCharts-NRG
